# App de Cobros Parciales para Bares y Restaurantes

Aplicación para gestionar pagos parciales en bares y restaurantes mediante QR.

## Características
- ✅ Acceso mediante código QR
- ✅ Selección de menú
- ✅ Pagos parciales
- ✅ Múltiples locales
- ✅ Web y Móvil

## Stack Tecnológico
- Frontend: React
- Backend: Node.js + Express
- Base de datos: Firebase
- Despliegue: Vercel + Firebase
